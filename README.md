# Instruction

## Instruction how to install project

1. Install NodeJS
2. Pull project from repository
3. Go to project root folder
4. npm install

## How to run

1. To run module:
    - To run in development mode
        - npm run dev
    - To run in production mode
        - npm run start

## How to build

1. To build module:
    - npm run build

All build files are in the folder "./dist".
